## 📌 Project Overview
This project predicts whether a person has diabetes using medical features.

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib

## 📊 Model Used
- Logistic Regression

## 🎯 Accuracy
Achieved 100% accuracy on test dataset.

- dataset.csv
